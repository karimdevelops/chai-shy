## About The Project

![home](./images/home.png)

Bringing back the traditional drink embraced in many cultures: chai (tea). This restaurant project is not just any ordinary site. It has many features combined with the perfect drink listed below. Chai Shy not only has the typical tea flavor but it offers multiple flavors and also a variety of other food items. 

Here's why: 

- A satisfying user experience because of fun animations.
- From managing profile to viewing order history.
- An admin dashboard with live stats.
- Live custom support chat.

While some features may look ordinary, they are not until you check them out!

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* pnpm

### Installation

1. Clone the repo
   ```sh
   https://github.com/karimdevelops/chai-shy/
   ```
3. Install packages
   ```sh
   pnpm install
   ```
4. Enter necessary variables in .env
   ```env
    # client/.env
    PROXY_TARGET="SERVER LINK"

    # api/.env
    SECRET="SERVER SECRET"
    PORT="SERVER PORT"
   ```

5. Run the server
```sh
pnpm run dev
```

6. Run the client
```sh
pnpm run client
```
